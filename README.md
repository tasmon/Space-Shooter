# Space Shooter for CloudPhone

A feature‑rich arcade shooter game designed for CloudPhone devices (240×320 resolution).  
Features include:
- **T9 Keypad Controls**: 2 = Up, 8 = Down, 4 = Left, 6 = Right, 0 = Pause.  
- **Gameplay**: Smooth scrolling, enemy waves, score tracking, and lives system.  
- **Settings Page**: Difficulty levels (Easy, Medium, Hard) and sound toggle.  
- **Help Page**: Quick guide to controls.

## Add to CloudPhone
1. Go to [CloudPhone Developer Page](https://www.cloudfone.com/my).  
2. Select **Add Widget**.  
3. Paste this Start URL:  
   `https://tasmon.github.io/Space-Shooter`  
4. Upload icons if required.  
5. Save and refresh — **Space Shooter** will appear on your CloudPhone.
